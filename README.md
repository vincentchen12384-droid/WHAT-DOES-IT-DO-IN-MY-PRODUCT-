# WHAT-DOES-IT-DO-IN-MY-PRODUCT-

最近在关注护肤品的时候，我发现一个很常见的问题：

当我们看到配料表里一长串复杂的成分名称时，
即使去网上搜索，也常常会遇到信息过多、说法不一的情况，
最后反而更困惑，不知道该相信什么。

👉 这个项目就是为了解决这个问题。

---

## 我在做什么？

这里不会简单告诉你：

❌ “这个成分有毒”
❌ “这个成分一定安全”

而是尝试回答一个更实际的问题：

👉 **它在产品里是干嘛的？**

---

## 项目目标

* 帮你理解成分的作用，而不是被名称吓到
* 解释常见的误解（比如“听起来危险 ≠ 实际危险”）
* 提供更接近真实使用场景的理解方式

---

## 适合谁？

* 看不懂配料表的人
* 查过资料但越查越乱的人
* 想更理性看待成分的人

---

## 一句话总结

👉 不是告诉你该不该买，而是让你知道你在用什么
